## 🚀 Быстрый старт

### Установка

1. **Скачайте последнюю версию APK:**
   ```bash
   https://github.com/VladosFlexXx/imes_app/releases/latest
   ```

2. **Разрешите установку из неизвестных источников:**
   - Настройки → Безопасность → Неизвестные источники

3. **Установите приложение и авторизуйтесь:**
   - Используйте свои данные от ЭИОС
   - Cookie не передаются третьим лицам

> ⚠️ **Внимание:** Приложение находится в **BETA**-версии. Возможны баги и изменения.

## 🐞 Сообщить о проблеме

Нашли баг? Помогите сделать приложение лучше!

### Шаги для репорта:

1. 📱 Откройте **Профиль → Диагностика**
2. 📦 Соберите отчёт (логи + кеш + состояние)
3. 📤 Отправьте файл или создайте [Issue](https://github.com/VladosFlexXx/imes_app/issues/new)

> 💡 **Совет:** Диагностический отчёт помогает быстрее найти и исправить ошибку!
